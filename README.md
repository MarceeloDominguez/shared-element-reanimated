## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/shared-element-reanimated.git
```

Go to the project directory

```bash
  cd shared-element-reanimated
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run android
```
